# debug

# 使用方法 
引用debug.js,尽量放到html的最后。
  ```code 
  <script src='debug.js'></script>
  ```
  在第一个tab你可以看到错误信息，第二个tab可以执行一些代码，让你在手机中也可以进行获取代码执行的错误信息
